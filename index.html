introduction

  • JavaScript is a high-level, interpreted programming language primarily used for client-side web development. It allows developers to create dynamic and interactive web pages.
  • Initially designed for adding interactivity to web pages, JavaScript is now widely used for both front-end and back-end development

    • what you should already know

  • HTML: Structure and content of web pages.
  • CSS: Styling and layout of web pages.
  • Programming concepts: Familiarity with variables, data types, and basic control flow in other programming languages can also be helpful.

    • variables in javascript

  • var: Declares a variable globally scoped or function scoped.
  • let: Declares a block-scoped variable, which means it is limited to the block, statement, or expression in which it is used.
  • const: Declares a block-scoped variable like let, but its value can't be reassigned once set.

    • Example

    var name = "John";
let age = 30;
const PI = 3.14;

    data types in javascript

  • JavaScript supports several primitive data types:
  • String: Sequence of characters, e.g., "Hello, World!".
  • Number: Numeric data, e.g., 42 or 3.14.
  • Boolean: Logical data, true or false.
  • Undefined: Variable that has been declared but not assigned a value.
  • Null: Represents an intentional absence of any object value.
  • Symbol: Unique and immutable value, introduced in ECMAScript 6 (ES6).
  • BigInt: Represents whole numbers larger than 2^53 - 1, introduced in ES11.

    • Example

    let message = "Hello";
let count = 10;
let isFound = true;
let height = undefined;
let empty = null;

    conditionls in javascript

  • Conditionals in JavaScript are used to make decisions in your code based on specific conditions. The main conditional statements are:
  • if statement: Executes a block of code if a specified condition is true.
  • else statement: Executes a block of code if the same condition is false.
  • else if statement: Allows you to specify a new condition if the first condition is false.
  • switch statement: Evaluates an expression, matching the expression's value to a case clause, and executes statements associated with that case.

    • Example:

    let hour = 15;

if (hour < 12) {
console.log("Good morning!");
} else if (hour < 18) {
console.log("Good afternoon!");
} else {
console.log("Good evening!");
